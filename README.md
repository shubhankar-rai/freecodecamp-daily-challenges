# 🧩 FreeCodeCamp Daily Challenges

A daily practice repo — one challenge at a time, building consistency and sharpening fundamentals across algorithms, data structures, and fullstack projects.

## 📁 Structure

```
freecodcamp-challenges/
├── [year-month]/      # Divided by the year and month
```

Each challenge lives in its own dated folder:

```
[year-month]/
└── [date]_[challenge-name]/
    ├── solution.js
    └── notes.md    ← approach, edge cases, learnings
```
